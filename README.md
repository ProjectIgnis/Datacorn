# Datacorn

## How to download
Get the latest release that matches your Operating System:
 * For Linux:
	- [`datacorn-qt6`](https://github.com/ProjectIgnis/Datacorn/releases/latest/download/datacorn-qt6) (Recommended, requires Qt 6.2.4 or later and glibc 2.35 or later)
	- [`datacorn`](https://github.com/ProjectIgnis/Datacorn/releases/latest/download/datacorn) (Requires Qt 5.9.5 or later and glibc 2.27 or later)
 * For Windows:
    - [`Datacorn-mingw-qt6.exe`](https://github.com/ProjectIgnis/Datacorn/releases/latest/download/Datacorn-mingw-qt6.exe) (Recommended, requires Windows 10 x64 or later)
    - [`Datacorn-mingw.exe`](https://github.com/ProjectIgnis/Datacorn/releases/latest/download/Datacorn-mingw.exe) (Second best recommended, requires Windows 7 or later)
    - [`Datacorn.exe`](https://github.com/ProjectIgnis/Datacorn/releases/latest/download/Datacorn.exe) (Requires Windows XP or later)
 * For MacOS:
    - [`Datacorn-macos-qt6.dmg`](https://github.com/ProjectIgnis/Datacorn/releases/latest/download/Datacorn-macos-qt6.dmg) (Recommended, universal binary, requires Mac OS 11 or later)
    - [`Datacorn-macos-qt5.dmg`](https://github.com/ProjectIgnis/Datacorn/releases/latest/download/Datacorn-macos-qt5.dmg) (Requires Mac OS 10.13 or later)
